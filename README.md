# Projetos com Aruduino UNO


Este repositório reúne **exercícios práticos e projetos com Arduino Uno**, desenvolvidos durante meu curso técnico em eletrônica. O objetivo era aplicar conceitos de **eletrônica analógica e digital, bem como microcontroladores**, criando soluções simples, funcionais e educativas.

</div>

---

## Principais Aprendizados

Ao longo desses projetos, desenvolvi habilidades em:
- Montagem e interpretação de circuitos eletrônicos
- Programação lógica com Arduino (C/C++)
- Leitura de sensores e trabalho com sinais digitais/analógicos
- Organização e documentação de projetos técnicos
